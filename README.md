# FPGA-Based-AHB-to-APB-Bridge-Using-Verilog-HDL-Incorporating-Reduced-Latency
FPGA-based AHB-to-APB Bridge implemented in Verilog HDL with a low-latency architecture. The design employs early address decoding, an optimized FSM, transaction overlap, and fine-grain clock gating to reduce write latency by 25%. Developed in Xilinx Vivado and validated on a Spartan-7 FPGA.
